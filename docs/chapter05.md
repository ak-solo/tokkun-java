# 5章 配列

## 基礎知識

### 配列の宣言

`型名[] 変数名 = new 型名[要素数];` で宣言します。インデックスは 0 から始まります。

```java
int[] numbers = new int[5];  // 要素数 5（インデックス 0〜4）
numbers[0] = 10;
numbers[4] = 50;
```

宣言と同時に初期値を設定することもできます。

```java
int[] primes = { 2, 3, 5, 7, 11 };  // 要素数 5
```

---

### 配列の走査

`for` ループでインデックスを使う方法と、拡張 `for`（foreach 相当）で要素を順に取り出す方法があります。

```java
int[] scores = { 80, 60, 90, 70 };

// インデックスで走査
for (int i = 0; i < scores.length; i++) {
    System.out.println(scores[i]);
}

// 拡張 for（インデックスが不要なとき）
for (int s : scores) {
    System.out.println(s);
}
```

> **注意:** 配列の要素数は `arr.length` です（文字列の `str.length()` と違い、括弧は不要です）。

---

### 逆順アクセス

インデックスを末尾から遡ることで逆順に処理できます。

```java
int[] arr = { 1, 2, 3, 4, 5 };
for (int i = arr.length - 1; i >= 0; i--) {
    System.out.print(arr[i] + " ");  // 5 4 3 2 1
}
```

---

### 2次元配列

`int[][]` で 2 次元配列を宣言します。

```java
int[][] matrix = new int[3][3];  // 3×3
matrix[0][0] = 1;
matrix[1][2] = 6;

// 二重ループで走査
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.printf("%3d", matrix[i][j]);
    }
    System.out.println();
}
```

---

### 動的な配列収集（ArrayList）

要素数が事前にわからないときは `ArrayList<Integer>` を使い、最後に `int[]` に変換します。

```java
import java.util.ArrayList;
import java.util.List;

List<Integer> evens = new ArrayList<>();
for (int n : new int[]{1, 2, 3, 4, 5, 6}) {
    if (n % 2 == 0) evens.add(n);
}
int[] result = evens.stream().mapToInt(Integer::intValue).toArray();  // {2, 4, 6}
```

---

### 配列のソート

`Arrays.sort` を使うと昇順に並び替えられます。**元の配列を変更する**ため、コピーが必要な場合は `Arrays.copyOf(arr, arr.length)` を使います。

```java
import java.util.Arrays;

int[] arr = { 5, 3, 8, 1 };
int[] copy = Arrays.copyOf(arr, arr.length);  // コピーを作成
Arrays.sort(copy);  // copy = {1, 3, 5, 8}（arr は変わらない）
```

---

### 配列の文字列化

`Arrays.stream` と `String.join` を組み合わせると配列の要素を区切り文字でつなげられます。

```java
import java.util.Arrays;
import java.util.stream.Collectors;

int[] numbers = { 1, 2, 3 };
String result = Arrays.stream(numbers)
    .mapToObj(String::valueOf)
    .collect(Collectors.joining(","));  // "1,2,3"
```
