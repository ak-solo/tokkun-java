# 2章 引数と戻り値

## 基礎知識

### メソッドに値を渡す（引数）

メソッドは**引数**を通じて値を受け取ることができます。引数はメソッドを呼び出す側から渡す「入力値」です。

```java
static int doubleIt(int x) {
    return x * 2;
}

System.out.println(doubleIt(5));  // → 10
System.out.println(doubleIt(3));  // → 6
```

引数を変えるだけで同じメソッドを何度でも使いまわせます。

---

### 引数を複数定義する

カンマ区切りで複数の引数を定義できます。

```java
static int sum(int x, int y) {
    return x + y;
}

System.out.println(sum(3, 7));    // → 10
System.out.println(sum(10, 20));  // → 30
```

---

### 引数の型

引数にも変数と同様に型を指定します。

| 型 | 渡せる値の例 |
|---|---|
| `int` | `10`, `-5`, `0` |
| `double` | `3.14`, `0.5` |
| `String` | `"hello"`, `"Java"` |

---

### 複数の値を返す（文字列結合）

複数の計算結果をまとめて返したいときは、カンマ区切りの文字列にまとめる方法があります。

```java
static String powers(int x) {
    return x + "," + (x * x) + "," + (x * x * x);
}

System.out.println(powers(2));  // → 2,4,8
System.out.println(powers(3));  // → 3,9,27
```

## 練習問題

### 問題 2-1

`String` 型の引数 `s` を受け取り、`s` をそのまま返すメソッドを実装しなさい。

---

### 問題 2-2

`int` 型の引数 `x` を受け取り、`x` をそのまま返すメソッドを実装しなさい。

---

### 問題 2-3

`int` 型の引数 `x` を受け取り、`x` の **1 乗・2 乗・3 乗** をカンマ区切りの文字列で返すメソッドを実装しなさい。

例: `x = 2` のとき `"2,4,8"` を返す

---

### 問題 2-4

`int` 型の引数 `x`、`y` を受け取り、以下の計算結果をそれぞれ返すメソッドを実装しなさい。

| メソッド名 | 内容 | 戻り値の型 |
|---|---|---|
| `problem2_4_Sum` | x と y の和 | `int` |
| `problem2_4_Difference` | x と y の差（x − y） | `int` |
| `problem2_4_Product` | x と y の積 | `int` |
| `problem2_4_Division` | x ÷ y の結果（小数あり） | `double` |
| `problem2_4_Quotient` | x ÷ y の商（整数） | `int` |
| `problem2_4_Remainder` | x ÷ y の余り | `int` |

**ヒント:** 小数ありの除算は `(double)x / y`、整数の商は `x / y`、余りは `x % y` を使います。

---

### 問題 2-5

`int` 型の引数 `a`、`b` を受け取り、2 つの整数の平均値（整数）を返すメソッドを実装しなさい。

※ 小数点以下は切り捨ててよい。

**ヒント:** `int` 同士の `(a + b) / 2` は自動的に切り捨てになります。

---

### 問題 2-6

`int` 型の引数 `age`（年齢）を受け取り、生まれてからのおおよその日数を返すメソッドを実装しなさい。

※ 閏年は考慮せず、`年齢 × 365` で計算する。
