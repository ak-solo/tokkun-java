# 1章 表示・変数・演算

## 基礎知識

### プログラミングとは

コンピューターは、人間が命令を与えないと何もできません。**プログラム**とは、コンピューターへの命令を順番に並べたものです。そしてプログラムを書くことを**プログラミング**といいます。

```
命令1: x に 10 を入れろ
命令2: x を 2 倍にしろ
命令3: x の値を画面に表示しろ
→ 20 が表示される
```

コンピューターは上から順番に命令を実行します。順番が変わると結果も変わります。

プログラムは**プログラミング言語**という人間とコンピューターの共通言語で書きます。この教材では **Java**（ジャバ）というプログラミング言語を使います。

---

### 変数とは

**変数**とは、値を一時的にしまっておく「箱」のようなものです。箱には名前をつけておき、後から名前を使って中身を取り出せます。

```
┌─────────┐
│   10    │  ← 中身（値）
└─────────┘
     x        ← 箱の名前（変数名）
```

変数を使うと、同じ値を何度も書かずに済みますし、値が変わっても一か所を直すだけで対応できます。

たとえば、ゲームのスコアを変数 `score` に入れておけば、得点が増えるたびに `score` を更新するだけで済みます。

```java
int score = 0;   // 最初は 0 点
score = score + 10;  // 10 点加算 → score は 10 に
score = score + 5;   // さらに 5 点加算 → score は 15 に
```

---

### データ型とは

変数の箱には「どんな種類の値を入れるか」という**型（データ型）**が決まっています。整数を入れる箱、小数を入れる箱、文字を入れる箱…というように、種類ごとに使う型が異なります。

型を決める理由は次の通りです。
- コンピューターが値をどう扱えばよいか判断できる
- 誤った種類のデータを入れてしまうミスを防げる

たとえば「足し算」は数値同士なら計算できますが、文字列同士だと文字を繋げる操作（結合）になります。型があることでコンピューターは適切に処理を切り替えられます。

---

### 画面への出力

`System.out.println` は引数を表示して改行します。`System.out.print` は改行しません。

```java
System.out.println("Hello World");  // Hello World と表示して改行
System.out.print("Hello");          // Hello と表示（改行なし）
System.out.print(" World");         // Hello World となる
```

文字列の中に変数の値を埋め込むには、`+` で連結するか `String.format` を使います。

```java
int x = 10;
System.out.println("x=" + x);              // x=10 と表示
System.out.println("x=" + x + ",y=" + 5); // x=10,y=5 と表示
System.out.println(String.format("x=%d", x)); // String.format を使う方法
```

---

### 変数の宣言と代入

変数を使う前に、「この名前の箱を用意する」という宣言が必要です。これを**変数の宣言**といいます。Java では `型名 変数名;` と書きます。

```java
int x;          // int 型の変数 x を宣言（箱を用意する）
int y = 5;      // 宣言と同時に 5 を代入（箱を用意して 5 を入れる）
x = 10;         // x に 10 を代入（箱の中身を 10 にする）
```

`=` は「等しい」ではなく「右の値を左の変数に入れる」という意味です（**代入**）。数学の `=` とは意味が違うので注意しましょう。

```java
x = x + 1;  // 「x と x+1 が等しい」ではなく「x の値に 1 を足して x に入れる」
```

#### 主なデータ型

| 型名 | 内容 | 例 |
|---|---|---|
| `int` | 整数（−2,147,483,648 〜 2,147,483,647） | `42`, `-7` |
| `double` | 倍精度浮動小数点数 | `3.14`, `-0.5` |
| `String` | 文字列 | `"Hello"` |
| `boolean` | 真偽値 | `true`, `false` |

### 演算子

**演算子**は計算に使う記号です。`3 + 2` のような計算式を**式**といい、式を評価すると**値**が得られます。

| 演算子 | 意味 | 例 | 結果 |
|---|---|---|---|
| `+` | 加算 | `3 + 2` | `5` |
| `-` | 減算 | `3 - 2` | `1` |
| `*` | 乗算 | `3 * 2` | `6` |
| `/` | 除算（int 同士は整数除算） | `10 / 3` | `3` |
| `/` | 除算（double なら小数あり） | `10.0 / 3` | `3.333...` |
| `%` | 剰余（余り） | `10 % 3` | `1` |
| `Math.pow(x, n)` | 累乗（double を返す） | `Math.pow(2, 8)` | `256.0` |

> **注意:** `int` 型同士の `/` は整数除算になります（小数部切り捨て）。
> 小数を含む除算をしたい場合は、どちらかを `double` にする必要があります（例: `(double)x / y` または `10.0 / 3`）。

> **注意:** `Math.pow` は `double` を返すため、整数として使いたい場合は `(int) Math.pow(x, 2)` のようにキャストします。

## 練習問題

### 問題 1-1

`"Hello World"` という文字列を返すメソッドを実装しなさい。

**ヒント:** `return "Hello World";` と書くだけです。

---

### 問題 1-2

`int` 型の変数 `x` に数値 `11` を代入し、`"x=11"` という文字列を返すメソッドを実装しなさい。

**ヒント:** `"x=" + x` のように文字列結合を使いましょう。

---

### 問題 1-3

`int` 型の変数 `x`、`y` に数値 `13`、`17` を代入し、`"x=13,y=17"` という文字列を返すメソッドを実装しなさい。

---

### 問題 1-4

`int` 型の変数 `x` に `13` と `17` の和を代入し、`x` の値（整数）を返すメソッドを実装しなさい。

**ヒント:** `int x = 13 + 17;` のように書いて `return x;` します。

---

### 問題 1-5

`13` と `17` の積（整数）を返すメソッドを実装しなさい。ただし、**変数を使わずに** `return` の式の中で演算すること。

---

### 問題 1-6

次の手順で計算し、途中の値を `"21,10"` のようなカンマ区切りの文字列で返すメソッドを実装しなさい。

1. `int` 型の変数 `x` に `7` を代入する
2. `x` の値を 3 倍にして `x` に代入する
3. `x` を 2 で整数除算した値を求める
4. ステップ 2 の結果とステップ 3 の結果をカンマ区切りで返す

**ヒント:** `x` を 3 倍した後、`x + "," + (x / 2)` のように文字列結合でまとめて返せます。

---

### 問題 1-7

`int` 型の変数 `x`、`y` にそれぞれ `3`、`7` を代入し、`x` と `y` の値を**入れ替えて**、`"x=7,y=3"` という文字列を返すメソッドを実装しなさい。

**ヒント:** `x` と `y` 以外に、もうひとつ変数が必要です。

---

### 問題 1-8

`int` 型の変数 `x`、`y` に `19`、`23` を代入し、その積を変数 `z` に代入して `z` の値（整数）を返すメソッドを実装しなさい。

---

### 問題 1-9

`int` 型の引数 `x` を受け取り、`x` を **2 倍・3 倍・4 倍** した結果をカンマ区切りの文字列で返すメソッドを実装しなさい。

例: `x = 3` のとき `"6,9,12"` を返す

---

### 問題 1-10

`int` 型の引数 `x` を受け取り、`x` の **1 乗・2 乗・3 乗** をカンマ区切りの文字列で返すメソッドを実装しなさい。

例: `x = 2` のとき `"2,4,8"` を返す

**ヒント:** 累乗は `(int) Math.pow(x, 2)` または `x * x` を使います。

---

### 問題 1-11

`int` 型の変数 `x`、`y` に `10`、`3` を代入し、`(double)x / y` の結果（`double`）を返すメソッドを実装しなさい。

**ポイント:** `int` 同士の `x / y` は整数除算になるので、`(double)` キャストが必要です。

---

### 問題 1-12

`int` 型の変数 `x` に `10 / 3`（整数除算）の結果を代入し、`x` の値（整数）を返すメソッドを実装しなさい。

**ポイント:** 問題 1-11 と比べると、`int` 同士の `/` と `(double)/` の違いがよくわかります。

---

### 問題 1-13

2 つの `int` 型引数 `x`、`y`（`x` > `y`）を受け取り、`x ÷ y` の **商と余り** をカンマ区切りの文字列で返すメソッドを実装しなさい。

例: `x = 10, y = 3` のとき `"3,1"` を返す（商 3、余り 1）

**ヒント:** `int` 同士の `/` が商、`%` が余りです。
