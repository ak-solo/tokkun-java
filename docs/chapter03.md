# 3章 分岐

## 基礎知識

### 分岐とは

プログラムは通常、上から順番に実行されます。しかし「点数が 60 点以上なら合格、そうでなければ不合格」のように、**条件によって実行する処理を変えたい**場面があります。これを**分岐**といいます。

```
　　　条件を判定
　　　　　↓
　　┌─────────┐
　　│ 条件が真？│
　　└─────────┘
　　  ↙         ↘
　true のとき  false のとき
　  処理 A      処理 B
```

分岐を使うことで、状況に応じた柔軟なプログラムが書けるようになります。

---

### boolean 型と条件式

**boolean 型**は `true`（真）か `false`（偽）の 2 値だけを持つ型です。`if` 文の条件部分には boolean の値が入ります。

```java
boolean isPass = true;
boolean isFail = false;
```

**条件式**（比較演算子を使った式）を評価すると boolean 値になります。

```java
int x = 10;
System.out.println(x > 5);   // → true
System.out.println(x > 20);  // → false
System.out.println(x == 10); // → true
```

boolean 型の変数に条件式の結果を入れておき、後で使うこともできます。

```java
boolean isPositive = (x > 0);
if (isPositive) {
    System.out.println("正の数です");
}
```

---

### if 文

条件によって処理を分岐するには `if` 文を使います。

```java
if (x > 0) {
    System.out.println("正の数です");
}
```

`else` を加えると、条件が偽のときの処理も書けます。

```java
if (x >= 60) {
    System.out.println("合格");
} else {
    System.out.println("不合格");
}
```

`else if` で 3 つ以上の分岐も表現できます。

```java
if (score >= 80) {
    System.out.println("優");
} else if (score >= 70) {
    System.out.println("良");
} else if (score >= 60) {
    System.out.println("可");
} else {
    System.out.println("不可");
}
```

---

### 比較演算子

条件式で使う演算子です。結果は必ず `boolean`（`true` か `false`）になります。

| 演算子 | 意味 | 例 | 結果（x=10 のとき） |
|---|---|---|---|
| `>` | より大きい | `x > 5` | `true` |
| `<` | より小さい | `x < 5` | `false` |
| `>=` | 以上 | `x >= 10` | `true` |
| `<=` | 以下 | `x <= 9` | `false` |
| `==` | 等しい | `x == 10` | `true` |
| `!=` | 等しくない | `x != 10` | `false` |

> **注意:** Java では変数への代入は `=`、比較は `==` と記号が違います。`if (x = 10)` と書くとエラーになります。

---

### 論理演算子

複数の条件を組み合わせるには論理演算子を使います。

| 演算子 | 意味 | 真になる条件 |
|---|---|---|
| `&&` | かつ | 左辺も右辺も `true` |
| `\|\|` | または | 左辺か右辺のどちらかが `true` |
| `!` | ではない | 元の値が `false` |

```java
int x = 5;
System.out.println(x > 0 && x < 10);  // → true（0より大きく10より小さい）
System.out.println(x < 0 || x > 3);   // → true（どちらかが成立）
System.out.println(!(x == 5));         // → false（x=5 を反転）
```

`&&` は左辺が `false` なら右辺を評価しません（**短絡評価**）。
`||` は左辺が `true` なら右辺を評価しません（短絡評価）。

```java
// x が 0 のとき、y / x はゼロ除算エラーになる
if (x != 0 && y / x > 1)  // x=0 なら右辺を評価しない → エラー回避
{
    System.out.println("OK");
}
```

### switch 文

変数の値に応じて多くの分岐があるときは `switch` が読みやすくなります。

```java
switch (month) {
    case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        System.out.println("大の月");
        break;
    case 2: case 4: case 6: case 9: case 11:
        System.out.println("小の月");
        break;
    default:
        System.out.println("そんな月はありません");
        break;
}
```

> **注意:** Java の `switch` では、各 `case` の末尾に `break` を書かないと次の `case` に処理が続きます（フォールスルー）。必ず `break` または `return` を書くようにしましょう。

---

### 偶数・奇数の判定

`%` 演算子（余り）を使います。2 で割った余りが 0 なら偶数、それ以外なら奇数です。

```java
if (x % 2 == 0) {
    System.out.println("偶数");
} else {
    System.out.println("奇数");
}
```

負の数も同様に判定できます（例: `-4 % 2 == 0` → 偶数）。
