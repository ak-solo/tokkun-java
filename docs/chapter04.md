# 4章 繰り返し

## 基礎知識

### 繰り返しとは

同じ処理を何度も書くのは大変ですし、回数が変わるたびにコードを書き直さなければなりません。**繰り返し（ループ）** を使うと、同じ処理を指定した回数や条件に応じて自動的に繰り返せます。

```java
// 繰り返しなし（100 回書くのは無理）
System.out.println(1);
System.out.println(2);
System.out.println(3);
// ...

// 繰り返しあり（何回でも対応できる）
for (int i = 1; i <= 100; i++) {
    System.out.println(i);
}
```

Java には主に 3 種類のループがあります。

| ループ | 使いどころ |
|---|---|
| `for` | 繰り返す回数が決まっているとき |
| `while` | 条件が満たされる間くり返すとき |
| `do/while` | 最低 1 回は実行して、その後条件を判定するとき |

---

### for ループ

回数が決まっているときは `for` ループを使います。`i` はループのたびに 1 ずつ増える**ループ変数**（カウンタ）です。

```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);  // 1, 2, 3, 4, 5
}
```

実行の流れ：

```
i = 1 → ブロック実行 → i = 2 → ブロック実行 → ... → i = 5 → ブロック実行 → 終了
```

増減幅を変えることもできます。負の値にすると逆順にもできます。

```java
// 3 刻み
for (int i = 0; i <= 8; i += 3) {
    System.out.println(i);  // 0, 3, 6
}

// 逆順
for (int i = 5; i >= 1; i--) {
    System.out.println(i);  // 5, 4, 3, 2, 1
}
```

---

### while ループ

繰り返す回数が事前にわからないときは `while` を使います。条件が `true` の間、ブロックを繰り返します。

```java
int n = 1;
while (n <= 5) {
    System.out.println(n);
    n++;
}
```

> **注意:** ループ変数を自分で更新しないと、条件がずっと `true` のまま「**無限ループ**」になります。`n++` を忘れずに書きましょう。

---

### do/while ループ

`do/while` ループはブロックを実行してから条件を判定するため、**最低 1 回は必ず実行**されます。

```java
int n = 0;
do {
    n++;
    System.out.println(n);
} while (n < 5);  // 条件が真の間くり返す
```

`while` との違いは、条件が最初から `false` だった場合です。

```java
int n = 10;

// while: 条件が最初から false → 1 回も実行されない
while (n < 5) {
    System.out.println(n);  // 実行されない
}

// do/while: 先に実行してから判定 → 必ず 1 回実行される
do {
    System.out.println(n);  // 10 が 1 回だけ表示される
} while (n < 5);
```

---

### break / continue

`break` でループを途中で抜け、`continue` で今回の繰り返しをスキップして次へ進みます。

```java
for (int i = 1; i <= 10; i++) {
    if (i == 6) break;          // 6 に達したらループ終了
    if (i % 2 == 0) continue;  // 偶数はスキップ
    System.out.println(i);      // 1, 3, 5
}
```

実行の流れ（`i = 4` のとき）：

```
i = 4 → 偶数 → continue → i = 5 へジャンプ
i = 5 → 奇数 → println(5)
i = 6 → break → ループ終了
```

### 累計・最大・最小

ループ前に**集計用の変数**を初期化しておき、ループ内で更新していくパターンです。

```java
int[] numbers = { 5, 3, 8, 1, 9 };
int sum = 0;
int max = numbers[0];  // 最大（先頭値で初期化）
int min = numbers[0];  // 最小（先頭値で初期化）

for (int n : numbers) {
    sum += n;
    if (n > max) max = n;
    if (n < min) min = n;
}
// sum=26, max=9, min=1
```

最大・最小の初期値には先頭要素を使います。`0` で初期化すると全要素が負のときに誤った結果になるためです。

---

### ネストしたループ（二重ループ）

ループの中にループを入れることができます。外側が 1 周するたびに内側が最初から最後まで実行されます。

```java
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.print(i * j + " ");
    }
    System.out.println();  // 行末で改行
}
// 出力:
// 1 2 3
// 2 4 6
// 3 6 9
```

合計の実行回数は「外側のループ回数 × 内側のループ回数」になります（この例では 3 × 3 = 9 回）。

---

### 改行を含む文字列

複数行の文字列を返すときは `System.lineSeparator()` で改行を挟みます。

```java
List<String> lines = new ArrayList<>();
lines.add("1行目");
lines.add("2行目");
lines.add("3行目");
String result = String.join(System.lineSeparator(), lines);
System.out.println(result);
// 出力:
// 1行目
// 2行目
// 3行目
```

---

### カンマ区切りの文字列を作る

`ArrayList` と `String.join` を組み合わせると便利です。

```java
import java.util.ArrayList;
import java.util.List;

List<String> parts = new ArrayList<>();
for (int i = 1; i <= 5; i++) {
    parts.add(String.valueOf(i));  // int を String に変換して追加
}
String result = String.join(",", parts);  // "1,2,3,4,5"
```

または StringBuilder を使う方法もあります。

```java
StringBuilder sb = new StringBuilder();
for (int i = 3; i <= 27; i += 3) {
    if (sb.length() > 0) sb.append(",");
    sb.append(i);
}
String result = sb.toString();  // "3,6,9,...,27"
```
